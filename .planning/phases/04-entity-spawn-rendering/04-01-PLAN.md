  <name>Task 1: Install pixi-filters and create entity rendering modules</name>
  <files>client/package.json, client/src/world/captureEntityTexture.ts, client/src/world/EntitySprite.ts</files>
  <action>
1. Install pixi-filters v6 for DropShadowFilter:
   ```bash
   pnpm --filter @crayon-world/client add pixi-filters
   ```

2. Create `client/src/world/captureEntityTexture.ts`:
   - Export function `captureEntityTexture(app: Application, strokeContainer: Container): Texture`
   - Use `app.renderer.generateTexture()` with `clearColor: [0, 0, 0, 0]` for transparent background
   - Compute frame from `strokeContainer.getBounds()` to avoid empty space (same approach as exportPng.ts)
   - Use `resolution: 1` and `antialias: true`
   - Return the RenderTexture directly (not a Sprite — EntitySprite handles that)

3. Create `client/src/world/EntitySprite.ts`:
   - Export function `buildEntityContainer(texture: Texture, profile: EntityProfile, app: Application): Container`
   - Create a parent Container that owns: sprite, label, shadow
   - Create Sprite from texture with `anchor.set(0.5, 0.5)`
   - Scale sprite down: `const scaleFactor = 0.2` (roughly 1:5). Clamp so minimum dimension is 30px for readability
   - Apply `DropShadowFilter` from `pixi-filters` to sprite: `{ blur: 3, distance: 6, alpha: 0.35 }`
   - Create PixiJS v8 Text label: `new Text({ text: profile.name, style: new TextStyle({ fontSize: 12, fill: '#333333', fontWeight: 'bold' }) })`
   - Position label: `label.anchor.set(0.5, 1)`, `label.y = -(sprite.height / 2) - 6` (6px gap above scaled sprite top — sprite.height already reflects scale in v8)
   - Set `entity.eventMode = 'static'` and `entity.cursor = 'pointer'` for hover events (used by Plan 02)
   - Add fade-in: set `entity.alpha = 0`, then use `app.ticker.add` with callback receiving `Ticker` instance, increment elapsed via `ticker.deltaMS`, set `entity.alpha = Math.min(1, elapsed / 300)`, remove ticker callback when alpha >= 1
   - Return the Container

   IMPORTANT: PixiJS v8 Ticker callback receives a Ticker instance, NOT a number. Use `ticker.deltaMS` for milliseconds. Do NOT use the argument as a raw delta number.
   IMPORTANT: PixiJS v8 Text uses object params: `new Text({ text, style })`, NOT `new Text(text, style)`.
   IMPORTANT: Only Containers can have children in v8, not Sprites. The entity must be a Container.
  </action>

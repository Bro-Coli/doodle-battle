  <files>client/src/world/EntityTooltip.ts, client/src/world/EntitySprite.ts, client/src/style.css</files>
  <action>
1. Create `client/src/world/EntityTooltip.ts`:
   - Export functions `showTooltip(profile: EntityProfile, x: number, y: number): void` and `hideTooltip(): void`
   - Follow project pattern (RecognitionOverlay): create element on show, remove on hide, no persistent DOM nodes
   - Create a single `div` with class `entity-tooltip`, positioned absolutely via `style.left` / `style.top`
   - Tooltip content in plain text format (locked decision):
     ```
     Wolf — Walking
     Traits: predatory, pack hunter
     Role: Apex predator
     ```
   - Format: `${profile.name} — ${profile.archetype.charAt(0).toUpperCase() + profile.archetype.slice(1)}`
   - Then `Traits: ${profile.traits.join(', ')}`
   - Then `Role: ${profile.role}`
   - Use line breaks (`<br>`) or separate divs for each line
   - Position tooltip slightly offset from pointer (e.g., +12px right, +12px down) to avoid flickering
   - Track the active tooltip element in module-level variable; if `showTooltip` called while one exists, update position and content

2. Update `client/src/world/EntitySprite.ts`:
   - Import `showTooltip` and `hideTooltip` from `./EntityTooltip`
   - Add `profile` parameter to `buildEntityContainer` (already there from Plan 01)
   - After setting `entity.eventMode = 'static'`, wire hover events:
     ```typescript
     entity.on('pointerover', (e) => {
       const canvasBounds = app.canvas.getBoundingClientRect();
       showTooltip(profile, e.global.x + canvasBounds.left, e.global.y + canvasBounds.top);
     });
     entity.on('pointerout', () => hideTooltip());
     ```
   - Also wire `pointermove` on the entity to update tooltip position as cursor moves within entity bounds

3. Add tooltip CSS to `client/src/style.css`:
   - `.entity-tooltip`: position fixed, background white, border 1px solid #ccc, border-radius 6px, padding 8px 12px, font-size 13px, line-height 1.4, box-shadow 0 2px 8px rgba(0,0,0,0.15), pointer-events none (tooltip doesn't intercept mouse), z-index 1000, max-width 250px
   - First line (name + archetype) bold
  </action>

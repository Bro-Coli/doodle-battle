**Create `client/src/network/` directory** and `ColyseusClient.ts`:

```typescript
// client/src/network/ColyseusClient.ts
import { Client, Room } from '@colyseus/sdk';

const COLYSEUS_URL = 'http://localhost:3001';

let client: Client | null = null;

export function getClient(): Client {
  if (!client) {
    client = new Client(COLYSEUS_URL);
  }
  return client;
}

export async function connectToRoom(roomName: string = 'game_room'): Promise<Room> {
  const c = getClient();
  const room = await c.joinOrCreate(roomName);
  return room;
}
```

Use `http://localhost:3001` directly (not `ws://` — the SDK handles upgrade internally, per research Pitfall 6). This avoids needing a Vite WebSocket proxy for dev.

**Optionally update `client/vite.config.ts`** to add a WebSocket proxy for Colyseus if needed. Per research, direct connection to port 3001 works in dev without proxy, so only add this if direct connection fails:

**Add a temporary verification hook** to test the connection. Create a small function that can be called from the browser console or wired into a temporary button:

```typescript
// Add to ColyseusClient.ts as a named export for verification
export async function verifySync(): Promise<void> {
  const room = await connectToRoom();
  console.log('[Colyseus] Connected to room:', room.id);

  room.state.listen('tick', (value: number) => {
    console.log('[Colyseus] tick:', value);
  });

  // Auto-leave after 10 seconds
  setTimeout(() => {
    room.leave();
    console.log('[Colyseus] Left room after verification');
  }, 10000);
}
```

**Wire `verifySync` to `window`** for easy browser console testing. In `ColyseusClient.ts`, add at the bottom:
```typescript
// Temporary — for Phase 10 verification only
if (typeof window !== 'undefined') {
  (window as any).__verifyColyseusSync = verifySync;
}
```

This lets the human tester open browser console and run `__verifyColyseusSync()` to see tick values arriving.

Note: The `room.state.listen()` API is from @colyseus/sdk 0.17. If it doesn't work, try `import { Callbacks } from '@colyseus/sdk'` and use `Callbacks.get(room).listen('tick', cb)` instead (both patterns are documented in the research).

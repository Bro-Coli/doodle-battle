3. **Add startRoundBtn to disableAllToolbar():**
   Add `startRoundBtn.disabled = true;` inside the existing `disableAllToolbar` function.

4. **Add startRoundBtn to enableAllToolbar():**
   Do NOT blindly enable startRoundBtn here. After enabling other buttons, call a sync function. Add a new helper:
   ```typescript
   const syncStartRoundBtn = (): void => {
     startRoundBtn.disabled = worldStage.entityCount === 0 || worldStage.roundPhase !== 'idle';
   };
   ```
   Call `syncStartRoundBtn()` at the end of `enableAllToolbar()`.

5. **Call syncStartRoundBtn after entity spawn:**
   In the `submitRecognition` success callback (where `worldStage.spawnEntity` is called), call `syncStartRoundBtn()` after spawning. This enables the button once the first entity exists.

6. **Start Round click handler:**
   ```typescript
   startRoundBtn.addEventListener('click', () => {
     if (worldStage.roundPhase !== 'idle' || worldStage.entityCount === 0) return;
     startRoundBtn.disabled = true; // immediately disable to prevent double-click
     // Auto-switch to world view per locked decision
     if (!worldStage.inWorld) {
       worldStage.toggle();
       viewToggleBtn.textContent = 'Draw';
     }
     disableAllToolbar();
     void worldStage.startRound();
   });
   ```

7. **Wire onRoundPhaseChange callback:**
   After creating worldStage, set up the phase change callback:
   ```typescript
   worldStage.onRoundPhaseChange = (phase: RoundPhase) => {
     if (phase === 'idle') {
       // Round ended — auto-switch back to draw mode per locked decision
       if (worldStage.inWorld) {
         worldStage.toggle();
         viewToggleBtn.textContent = 'World';
       }
       enableAllToolbar();
       syncStartRoundBtn();
     }
     // During analyzing and simulating: toolbar stays disabled (already disabled from click handler)
     // viewToggle: disable during analyzing (spinner must stay visible), re-enable during simulating
     if (phase === 'analyzing') {
       viewToggleBtn.disabled = true;
     }
     if (phase === 'simulating') {
       viewToggleBtn.disabled = false; // player can peek during simulation
     }
   };
   ```

8. **Append startRoundBtn to toolbar** — insert it after viewToggleBtn:
   ```typescript
   toolbar.appendChild(startRoundBtn);
   ```
   Place this line right after `toolbar.appendChild(viewToggleBtn)`.

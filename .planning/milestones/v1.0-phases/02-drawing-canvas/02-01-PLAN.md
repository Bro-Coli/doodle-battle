1. Install perfect-freehand:
   ```bash
   pnpm --filter @crayon-world/client add perfect-freehand
   ```

2. Create `client/src/drawing/StrokeRenderer.ts`:
   - Export `THICKNESS_PRESETS` object with three keys: thin (size:4, thinning:0.6), medium (size:8, thinning:0.5), thick (size:16, thinning:0.4). All share smoothing:0.5, streamline:0.4, simulatePressure:true.
   - Export `ThicknessPreset` type as keyof typeof THICKNESS_PRESETS.
   - Export `renderStroke(gfx: Graphics, points: [number, number][], preset: ThicknessPreset): void` function:
     - Calls `getStroke(points, THICKNESS_PRESETS[preset])` from perfect-freehand
     - If outline has fewer than 2 points, return early
     - Calls `gfx.clear()`, then builds path using `gfx.moveTo` for first point, `gfx.quadraticCurveTo` for smooth curve through outline points (using midpoints as in research Pattern 2), then `gfx.fill({ color: 0x000000 })`

3. Create `client/src/drawing/UndoStack.ts`:
   - Import `Graphics` and `Container` from pixi.js
   - Export class `UndoStack` with:
     - Private `strokes: Graphics[]` array
     - Private `container: Container` (set via constructor)
     - `push(gfx: Graphics): void` -- adds to array and calls `container.addChild(gfx)`
     - `undo(): boolean` -- pops last, calls `container.removeChild(gfx)` then `gfx.destroy()`, returns true. Returns false if empty.
     - `clear(): void` -- destroys all Graphics, empties array, calls `container.removeAllChildren()`
     - `get isEmpty(): boolean` -- returns `strokes.length === 0`
     - `onChange: (() => void) | null` property -- called after push/undo/clear if set (for button state sync)

4. Create `client/src/drawing/DrawingCanvas.ts`:
   - Import Application, Graphics, Container from pixi.js
   - Import renderStroke and ThicknessPreset from StrokeRenderer
   - Import UndoStack from UndoStack
   - Export class `DrawingCanvas` with:
     - Constructor takes `app: Application`
     - Creates a white-filled Graphics rectangle as the drawing region, centered on stage. Dimensions: `Math.min(app.screen.width * 0.7, 680)` wide, `Math.min(app.screen.height * 0.75, 560)` tall. Position centered with 28px top offset for toolbar clearance.
     - Adds subtle border: `.stroke({ color: 0xd0d0d0, width: 1 })` on the region rect
     - Sets `region.eventMode = 'static'` and `region.cursor = 'crosshair'`
     - Creates a `Container` as child of the region for stroke Graphics
     - Creates an `UndoStack` with the stroke container
     - Private `currentPoints: [number, number][]` for the active stroke
     - Private `liveGraphics: Graphics` for the in-progress stroke (reused, cleared each frame)
     - Private `activePreset: ThicknessPreset` defaulting to 'medium'
     - `pointerdown` on region: if already drawing, ignore. Set drawing flag, reset currentPoints, create fresh liveGraphics added to stroke container, record first point (position relative to region using `e.getLocalPosition(region)`)
     - `globalpointermove` on `app.stage`: if not drawing, return. Append point (clamped to region bounds), call `renderStroke(liveGraphics, currentPoints, activePreset)`
     - `pointerup` and `pointerupoutside` on `app.stage`: if not drawing, return. Clear drawing flag. If currentPoints has >= 2 points, commit: create a new Graphics, call renderStroke on it, push to undoStack, then remove liveGraphics from container and destroy it. If < 2 points (just a click), still render a dot by pushing a duplicated point and committing.
     - Public `undo(): void` -- delegates to undoStack.undo()
     - Public `clear(): void` -- delegates to undoStack.clear()
     - Public `get isEmpty(): boolean` -- delegates to undoStack.isEmpty
     - Public `setThickness(preset: ThicknessPreset): void` -- sets activePreset
     - Public `get undoStack(): UndoStack` -- exposes for onChange wiring
     - Public `get strokeContainer(): Container` -- exposes for PNG export (Plan 02)
     - Public `get region(): Graphics` -- exposes for PNG export bounds
  </action>
